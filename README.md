# Homebrew Tap

Homebrew formulas for developer tools by [@LyricalString](https://github.com/LyricalString).

## Install

```sh
brew tap LyricalString/tap
brew install clide
brew install gale
```

## Packages

| Formula | Description |
|---------|-------------|
| [clide](https://github.com/LyricalString/clide) | High-performance drop-in replacement for Claude Code's terminal |
| [gale](https://github.com/LyricalString/gale) | Extremely fast CSS linter, drop-in replacement for Stylelint |
